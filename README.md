# s209-wireframe

AX-WMS 프론트엔드 프로토타입 저장소입니다.

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui 스타일 컴포넌트

## Run

```bash
cd apps/web
pnpm install
pnpm run dev
```

`npm`도 실행은 가능하지만, 한 번 설치를 시작했다면 같은 폴더에서는 `npm`과 `pnpm`을 섞어 쓰지 않는 것을 권장합니다. 패키지 매니저를 바꿔야 할 때는 `apps/web/node_modules`를 지우고 다시 설치하는 편이 안전합니다.
